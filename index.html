Get started

Three ways in

# 1. Download a prebuilt binary for your platform
# → see /download.html

# 2. Or build from source
git clone https://github.com/purcwix/novac.git
cd novac && cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release -j4

# 3. Run the REPL
./novac
# 3.5. (Optional) add to your PATH ./novac init PATH # and reload your terminal/VScode
