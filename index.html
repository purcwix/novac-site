Systems scripting language

A language and VM,
built from the ground up.

Novac is a scripting language with its own lexer, parser, and bytecode VM — plus a native FFI bridge for calling into C libraries and Windows COM directly from script.

Why Novac

Owned end to end

Custom C++ VM

A bytecode virtual machine written from scratch, with a value-type core built for primitive performance rather than pointer-chasing.

Native FFI

Call directly into C libraries via libffi — including native callbacks, variadic calls, and struct/union marshalling by value.

Windows COM interop

Script against COM objects directly, including WebView2, without hand-writing bindings for every interface.

A real REPL

Raw-mode line editing, arrow-key history, and built-in .edit / .time / .save commands.

Batteries included

A growing standard library — string scanning, lexing, string building, and an Std.LLM wrapper around llama.cpp.

nv+ package manager

Publish and install Novac packages through a GitHub PR–based registry workflow, no central server required.

Get started

Three ways in

# 1. Download a prebuilt binary for your platform
# → see /download.html

# 2. Or build from source
git clone https://github.com/purcwix/novac.git
cd novac && cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release -j4

# 3. Run the REPL
./novac
