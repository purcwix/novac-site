Scripting language

A language and executor,
built from the ground up.

Novac is a scripting language with its own lexer, parser, and C++ executor — plus a native library bridge, Windows GUI support, async/await, fibers, generators, and a package manager.

Get started

Three ways in

# 1. Download a prebuilt binary
#    → see /download.html

# 2. Build from source
git clone https://github.com/purcwix/novac.git
cd novac
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release -j4

# 3. Run the REPL
./novac
