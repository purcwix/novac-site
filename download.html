Build from source

Every platform, always up to date

git clone https://github.com/purcwix/novac.git
cd novac
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release -j4

Requires CMake 3.20+ and a C++20 compiler. See the build documentation for platform-specific notes.
